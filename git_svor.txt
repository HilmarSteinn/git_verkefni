#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd
change directory
	cd ~
fer í user folder
	ls
ls sýnir lista um allar möppur og fila í directoryinu
	pwd
prints working directory
	mkdir
makes a directory

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone
clonar file eða folder
	git log
sýnir log um commands sem hafa verið notaðar
	git status
sýnir status verkefnis
	git diff
git-diff - Show changes between commits, commit and working tree
	git checkout
git-checkout - Switch branches or restore working tree files

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 
git diff shows changes between commits, commit and working tree
	b) git diff --staged 
git staged er synonim fyrir --cached and shows changes that have been staged
	c) git diff commit1 commit2
